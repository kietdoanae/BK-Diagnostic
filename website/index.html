Capabilities

Full Diagnostic Feature Set

From real-time data streaming to fault code analysis and actuator testing

Real-Time Live Data

Displays RPM, vehicle speed, engine temperature, engine load, and 15+ sensor parameters continuously updated through an animated gauge interface.

Read & Clear Fault Codes (DTC)

Scan and decode OBD2 fault codes (P/C/B/U codes) from all ECUs. Clear DTCs and reset the Check Engine warning light in a single action.

ECU & VIN Information

Read vehicle VIN, ECU software version, and calibration numbers from modules: engine, transmission, ABS, airbag, and BCM.

Data Graph & Logger

Plot real-time graphs for multiple parameters simultaneously. Log data to CSV files for offline analysis after a test drive.

Actuator Tests

Activate and test actuators: EGR valve, fuel pump, electric fan, ABS system — via UDS Mode 31 (Routine Control).

Raw Frame Monitor

View raw CAN bytes side-by-side with decoded results — a debug tool for engineers to verify the correctness of the data decoding pipeline. Admin only

Supported Protocols

  • OBD2 Mode 01 — Live Data (standard)
  • OBD2 Mode 03/04 — DTC Read/Clear
  • Ford UDS Mode 22 — Proprietary PIDs
  • More OEMs (Toyota, Hyundai…)

Ford — Proprietary PIDs

  • Turbo Boost (DID 0x1046)
  • DPF Pressure (DID 0x1030)
  • EGR Rate (DID 0x2EF1)
  • Trans. Temp (DID 0x1940)
